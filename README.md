# 🚀 Portafolio Interactivo — Jenifer Gonzalez

> App desplegada en Render → **[Ver portafolio en vivo](https://TU-LINK.streamlit.app)**

## Correrlo localmente

```bash
# 1. Clonar el repo
git clone https://github.com/Jenifer-Gonzalez-DS-QA/portfolio-jenifer-gonzalez.git
cd portfolio-jenifer-gonzalez

# 2. Crear entorno virtual
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. Instalar dependencias
pip install -r requirements.txt

# 4. Ejecutar
python app.py
```

La app abre en `http://localhost:8501`
